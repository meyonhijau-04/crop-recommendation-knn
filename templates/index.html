Sistem Rekomendasi Tanaman

Masukkan kondisi lahan Anda dan sistem akan merekomendasikan tanaman yang paling cocok menggunakan algoritma K-Nearest Neighbors (KNN).

Saya Petani Saya Peneliti

Pilih Cara Input Data

Tersedia tiga cara untuk mendapatkan rekomendasi tanaman

Mode Petani
Jawab beberapa pertanyaan sederhana tentang kondisi lahan Anda. Tidak perlu mengetahui nilai teknis seperti N, P, K, atau pH.
Mulai
Mode Ahli
Masukkan nilai teknis secara langsung seperti kadar Nitrogen, Fosfor, Kalium, pH tanah, suhu, kelembapan, dan curah hujan.
Mulai
Upload CSV
Upload file CSV berisi banyak data lahan sekaligus. Sistem akan memproses semua data dan menampilkan hasil rekomendasi dalam tabel.
Mulai

Cara Kerja Sistem

Empat langkah sederhana untuk mendapatkan rekomendasi tanaman

01
Pilih Mode Input
Pilih Mode Petani jika tidak mengetahui nilai teknis, atau Mode Ahli jika memiliki data laboratorium.
02
Isi Data Lahan
Jawab pertanyaan sederhana tentang kondisi lahan Anda atau isi nilai N, P, K, pH, suhu, kelembapan, dan curah hujan.
03
Proses KNN
Algoritma KNN membandingkan data lahan Anda dengan 2200 data referensi untuk menemukan kecocokan terbaik.
04
Lihat Hasil
Sistem menampilkan rekomendasi tanaman beserta tingkat kepercayaan, informasi lengkap, dan tips penanaman.
